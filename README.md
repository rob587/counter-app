Esercizio 3 — Counter con limite

Un bottone reset che riporta il contatore a 0
Il numero diventa rosso quando è negativo, verde quando è positivo, nero quando è 0
Il bottone − si disabilita (opacità 0.3) quando il contatore arriva a -10
Il bottone + si disabilita quando arriva a 10

Concetti: stili condizionali, useState, logica con Pressable
